# PRISM-page

Official project page for **Fabric Image Demoiréing Benchmark from Synthesis to Restoration** (ECCV 2026).

The page is a static GitHub Pages site:

- `index.html` contains the project page content.
- `assets/site.css` contains the responsive styling.
- `assets/figures/` contains web-optimized figures exported from the paper.
- `.github/workflows/pages.yml` deploys the repository root to GitHub Pages.

After pushing to `main`, enable GitHub Pages in the repository settings and select **GitHub Actions** as the source.
